# Objectives
1. User sign-up  
2. ==login== with **Oauth2** flow  
3. Image Management  
    --Upload Image: Users should be able to upload images
    --Transform Image: Allow users to be able perform image transformations
    these transformations are:
    Resize, Crop, Rotate, Watermark, Flip, Mirror, Compress, Change format, Apply
    filters 
    --Retrieve Image: Users should to be able to retrieve a saved image in different
    formats
    --List Images: list all uploaded images by users and metadata
4. endpoints
    user multipart / form header to upload images
    -- /login, /images

5. cache images
